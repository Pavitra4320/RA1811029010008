Important Concepts :

Topic 1:Introduction to the Data Structures

Data Structures is a particular way of organizing data in a computer so that it can be used effectively. For example, we can store a list of items having the same data-type using the array data structure.

A) Data Structure Operations

The data in the data structures are processed by certain operations. The particular data structure chosen largely depends on the frequency of the operation that needs to be performed on the data structure.The operations include:

  • Traversing
  • Searching
  • Insertion
  • Deletion
  • Sorting
  • Merging

B) Algorithms

Data Structure Algorithms Basics:

  • Search − Algorithm to search an item in a data structure.
  • Insert − Algorithm to insert item in a data structure.
  • Update − Algorithm to update an existing item in a data structure.
  • Delete − Algorithm to delete an existing item from a data structure.

C) Time Complexity

The time complexity of an algorithm represents the number of steps it has to take to complete. The space complexity of an algorithm represents the amount of memory the algorithm needs in order to work. The time complexity of an algorithm describes how many steps an algorithm needs to take with respect to the input.

D) Asymptotic Notation

Asymptotic notations are mathematical tools to represent time complexity of algorithms for asymptotic analysis. The following 3 asymptotic notations are mostly used to represent time complexity of algorithms:

  • Theta Notation
  • Big O Notation
  • Omega Notation

E) Mathematical Notations and Functions

  2. Remainder Function (Modular Arithmetic)
  3. Integer and Absolute Value Functions
  4. Summation Symbol
  5. Factorial Function
  6. Exponents and Logarithms
  7. Permutations
